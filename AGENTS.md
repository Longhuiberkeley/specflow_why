# AGENTS.md

## Welcome, AI Agents!

This repository (`specflow_why`) is a presentation website built with **Slidev**. It is meant to be a 10-minute sharing session with a 5-minute Q&A to explain the "Why" and "How" of the SpecFlow framework.

### Guidelines for AI Agents

1. **Content First**: This is a Slidev presentation. The primary file is `slides.md`. If the user asks to modify the presentation, edit `slides.md`.
2. **Cute & Playful Theme**: Maintain a clean, bright, and playful aesthetic using Markdown frontmatter (e.g., pastel colors, rounded corners via inline styles or CSS, fun emojis).
3. **Keep it Simple**: Do not introduce complex build tools, databases, or unnecessary configurations. It is just a static Slidev site.
4. **DO NOT** create a `GEMINI.md`. Use this `AGENTS.md` file as the system truth for context.
