# SpecFlow - Sharing Presentation

This is a cute, playfully designed Slidev presentation for the 10-minute talk (plus 5-minute Q&A) about **SpecFlow**.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the local presentation server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3030` in your browser.

## Editing Slides

The presentation content lives entirely in `slides.md`. 
Check the **Quick Starter** slide (Slide 2) for a guide on how to add and format new slides.

## Deployment

The presentation is automatically deployed to GitHub Pages via GitHub Actions (`.github/workflows/deploy.yml`) on every push to `main` branch.
